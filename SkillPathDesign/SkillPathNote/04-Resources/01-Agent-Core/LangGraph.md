## LangChain Tools
LangChain wraps tools into a standard interface.
Agents can be given a list of tools and decide
which ones to use based on the task.

## In SkillPath
- **Tavily Search** → Find learning resources for each node
- **YouTube API** → Find video tutorials
- **OpenAI** → Generate quiz questions

## Key Insight
Tool calling is what separates an Agent from a Chatbot.
A chatbot answers. An Agent acts.
