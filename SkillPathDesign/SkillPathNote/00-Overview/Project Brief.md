# SkillPath · Project Brief

## One-liner
AI Learning Coach — plan roadmap, find resources, 
generate quizzes, remember weak points

## Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js + Express + LangGraph.js
- DB: MongoDB + Pinecone
- AI: OpenAI GPT-4o + Tavily Search
- Deploy: Vercel + Render

## Timeline
7 weeks, starting from Week 0 (Design)

## Current Status
🟡 Week 0 · Design Phase

## Core Features
- Smart learning roadmap generation
- Multi-source resource search (Tavily)
- Adaptive quizzes per learning node
- Long-term memory (Pinecone)
- Learning companion chat
- Progress dashboard

## Target Users
Developers and students learning new tech skills

## Differentiator
Not a chatbot — an Agent that remembers you,
plans for you, and pushes you to actually finish.
